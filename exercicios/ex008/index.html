Principais Formatações com HTML5

Negrito / Destaque

Este é um texto em negrito usando a tag <b>

A seguinte expressão está destacada, porque está usando a tag "strong".

Itálico / Ênfase

Para deixar um texto em itálico, podemos usar a tag "i".

Mas a forma correta de se aplicar uma itálico no HTML5 é usando a tag "em" - emphasis.

Marcação

Se quiser marcar um texto, basta usar a tag "MARK".

Se quiser marcar um texto com cor diferente, use o parâmetro style na tag "MARK".

Sobrescrito e Subescrito

Para sobrecrever um texto, como em: x5+7, use a tag "SUP".

Para subescrever um texto, como em: H2O, use a tag "SUB".
